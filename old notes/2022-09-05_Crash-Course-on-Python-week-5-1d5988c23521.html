object-oriented programming (OOP)

class: defined an idea
object/instance: an instance of the class
attributes: characteristics associated to a class(type)
methods: function associated to a class(type)

dir(<object>) # print all the attributes and methods
help(<object>) # return the documentation for the class
class <Name>: # names recommend start with capital letter
    pass      # placeholder in any block
class Name:
    attribute1 = ""
    attribute2 = "default"
name1 = Name
name1.attribute1 = "wah"
print(name1.attribute1, name1.attribute2) # "wah default"

dot notation(.): to access methods or attributes

classes and methods

class Name:
    attribute1 = 0
    def method1(self): #first parameter must be object it self
        self.attribute1*7
name1 = Name()
name1.attribute1 = 3
print(name1.method1()) # 21

instance variable: variables that have different values for different instances(object)

constructors (special methods)

class Name:
    def __init__(self, attribute1, attribute2): # always call <init>
        self.attribute1 = attribute1
        self.attribute2 = attribute2
name1 = Name(7, 3)
print(name1.attribute1) # 7

str (special methods)

class Name:
    attribute1 = 0
    def __str__(self): #when print(Name), call this method
        return "I say {} wah.".format(self.attribute1)
name1 = Name()
name1.attribute1 = 7
print(name1) # "I say 7 wah."

documentation (docstring)

class Name:
    """just a class""" # docstring
    def method1(self):
        """ just print wah""" # docstring
        print("wah")
help(Name) # show the docstring with class and method/function

classes and methods cheat sheet

inheritance

class A:
    def __init__(self, num1, num2):
        self.num1 = num1
        self.num2 = num2
class B(A):
    pass
class C(A):
    num1 = 10
b_object = B(3, 4)
print(b_object.num1) # 3
c_object = C(5, 6)
print(c_object.num1) #5
