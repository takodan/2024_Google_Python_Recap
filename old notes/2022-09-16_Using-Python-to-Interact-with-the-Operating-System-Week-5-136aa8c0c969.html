Test Concepts

white box test: the code is transparent. created alongside the development.
black box test: the code is opaque. can be created before the code.

integration test: a test with the overall system (best in the test environment).
regression test: a test that is written after identifying a bug.
smoke test: sanity check likes does program run or not.
load test: stressed test.
test suit: combine different tests.

test-driven development(TDD): creating the test before the code
