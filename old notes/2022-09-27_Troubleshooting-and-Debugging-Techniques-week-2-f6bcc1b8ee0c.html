expensive actions: can take a long time to complete.

  1. using the right data structures:
    list: access elements by position or iterate through all the elements.
    dictionary: use hashing: fast for searching keys.
  2. double-check creating structure copies is really needed.
  3. make sure to iterate elements only when you really need them to be.
  4. break out of the loop once you’ve found what you were looking for.
  5. check how long should the cache be recalculating.
  6. keeping results to avoid recalculating.
