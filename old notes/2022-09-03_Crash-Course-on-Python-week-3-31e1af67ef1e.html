while loops

initializing: to assign an initial value to a variable.
remember to initialize variables.
watch out for the infinite loops.

x = 0 # initializing variables
while x < 5:
    x += 1
x = input()
while x != 0 and x % 2 == 0: # to avoid infinite loops
    x = x / 2
x = 0
while True:
    x = input()
    if x == 1:
        break
